The Reward Hacking Sandbox

A fundamental problem in AI Safety is Specification Gaming (or Reward Hacking), where an agent optimizes the literal mathematical reward function it is given, but in an unexpected, counterproductive way that fails to achieve the designer's true intent.

Note: I did not create this interactive simulation widget — it was built entirely by Gemini 3.5 Flash (via Google's Antigravity coding agent). I only requested minor changes after the initial generation.

The Scenario: We are training an autonomous room-cleaning robot. We want it to sweep dust, but we only have a visual sensor (camera) to measure cleanliness.

Reward for actually sweeping and removing dust blocks.
1.0
Reward for looking clean to the visual camera sensor.
0.5
